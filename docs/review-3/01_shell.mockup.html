Annotation SCR-003
  • Zweck: globale Navigation zu Graph · Personen · Ereignisse · Timeline · Karte · Finder · Einstellungen.
  • Primär: Bereich wechseln (Klick auf Icon/Label).
  • Interaktion: ① schmale Rail = Default; ② Hover über Rail blendet Labels ein (gleitet aus), Verlassen klappt wieder ein; aktiver Bereich hervorgehoben; Icons mit Tooltip.
  • Zustände: kollabiert / hover-expanded / angeheftet / aktiv; A11y: per Tastatur fokussierbar, Labels auch bei Fokus sichtbar (nicht nur Hover).
  • Entschieden: 📌 Pin hält die Rail dauerhaft ausgeklappt (Zustand wird gespeichert).
